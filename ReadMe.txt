 Для вывода списка юзеров использовал шаблонизатор Handlebars, подключена в папке lib. Был опыт ее использования.

Сделал функционал:
- сообщение о том что пользователи не найдены
- добавление и удаление юзера
- юзеры получаются с помощью ajax запроса.
- калькулятор валют, курс получается с помощью запроса
- сделал валидацию для поля суммы для конвертации

Не делал:
-  валидацию полей для юзера и соответствующие сообщения
-  запросы для удаления/добавления юзера

Затраченное время 9 -10 часов.
